Markdown Converter

Please select some text and click the "Convert Selected" button OR click 'Convert All' to convert the entire document. Supported Markdown syntaxes are:

  • **bold** => bold
  • *italic* => italic
  • # (at start of line) => Heading 1
  • ## => Heading 2
  • ### => Heading 3
  • - (at start of line) => Bulleted list
  • `monospace` => monospace

Additional options are:

  • >> => Right align paragraph
  • || => Center align paragraph

Also, a code block marked with ``` will be converted to a pre-like table. 

```
code block
code block
```

will be converted to:

code block
code block
